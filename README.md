# 📖 About The Project

This project is a **modern Interactive Registration Form** designed with an elegant **Glassmorphism UI**, smooth animations, and real-time validation.

It demonstrates how HTML, CSS, and JavaScript can be combined to create a professional registration form with a responsive layout and interactive user experience.

---

# 🚀 Features

- 🎨 Modern Glassmorphism UI
- 🌈 Animated Gradient Background
- 📱 Fully Responsive Design
- 👤 Profile Image Upload & Preview
- 📊 Dynamic Progress Bar
- ✨ Floating Labels
- 🔐 Password Strength Meter
- 👁 Show/Hide Password
- ✅ Real-Time Form Validation
- 📧 Email Validation
- 📱 Phone Number Validation
- 🔒 Confirm Password Validation
- 🌍 Country Selection
- 🚻 Gender Selection
- 💼 Occupation Field
- ⭐ Skills Selection
- 📄 Terms & Conditions Checkbox
- 💾 Local Storage Support
- 🎉 Success Popup Notification
- ⚡ Smooth Hover & Focus Animations

# ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/sshashank011/Interactive-Registration-Form.git
```

### Navigate to project

```bash
cd Interactive-Registration-Form
```

### Run the project

Simply open

```
index.html
```

inside your browser.

No installation required.
